How can you help Tim?

For designers

Create contrast. When using multiple colors, make sure there is enough contrast between them. The color contrast ratio is a good indication to determine contrast.

  • Check the color contrast ratio between two colors
  • How to design (in Sketch, Photoshop and Web) for color blind users

Differentiate clustered elements not only on color. For example graphs or error messages can highly depend on color usage. Make sure colors are not the only way of distinction between different pieces of information.

For developers

Test on color contrast. Make sure your website/app keeps on having a good contrast.

  • Android: Test your app through the eyes of a person with colorblindness
  • iOS: Test on your iPhone/iPad with the accessibility features Grayscale and Invert Colours or use Color Oracle together with your emulator
  • Web: Test the color contrast with Tota11ly
